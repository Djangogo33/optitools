1. Qu'est-ce qu'un Cookie ?

Un cookie est un petit fichier texte stocké sur votre appareil par votre navigateur. Les cookies permettent aux sites web de :

  • Mémoriser vos préférences (langue, paramètres)
  • Vous reconnaître lors de futures visites
  • Analyser comment vous utilisez le site
  • Afficher des publicités pertinentes

Durée : Les cookies peuvent être stockés de quelques heures à plusieurs années selon leur type.

3. localStorage et sessionStorage

Qu'est-ce que localStorage ?

localStorage est similaire aux cookies mais peut stocker plus de données (5-10 MB). Utilisé pour :

  • Langue préférée (FR, EN, ES, etc.)
  • Paramètres de l'interface (mode sombre, taille texte)
  • Historique de consultation (si applicable)
  • Données d'authentification locales

Avantage : localStorage est plus rapide et persistant que les cookies traditionnels.

Comment Gérer localStorage ?

Dans la Console Navigateur (F12) :

// Voir toutes les données stockées:
console.log(localStorage);

// Supprimer une clé spécifique:
localStorage.removeItem('langue');

// Supprimer tous les localStorage:
localStorage.clear();

5. Gestion et Contrôle des Cookies

🖥️ Contrôle au Niveau du Navigateur

Pour désactiver les cookies dans votre navigateur :

  • Chrome : Paramètres → Confidentialité et sécurité → Cookies et autres données de site
  • Firefox : Paramètres → Vie privée → Cookies et données de site
  • Safari : Réglages → Confidentialité → Cookies et données de site
  • Edge : Paramètres → Confidentialité, recherche et services → Cookies et données de site

🔒 Mode Incognito/Privé

En utilisant le mode incognito/privé de votre navigateur :

  • Aucun cookie n'est stocké de manière persistante
  • localStorage est vidé à la fermeture
  • L'historique n'est pas enregistré

🛡️ Extensions de Blocage

Vous pouvez installer des extensions pour bloquer les trackers :

  • uBlock Origin : Bloqueur d'annonces et trackers
  • Privacy Badger : Blocage des trackers invisibles
  • Ghostery : Gestion des trackers
  • Disconnect.me : Sécurité et confidentialité

📡 Do Not Track (DNT)

Activer DNT dans votre navigateur (signal de préférence de confidentialité) :

  • Chrome : Paramètres → Confidentialité → Do Not Track
  • Firefox : Paramètres → Vie privée → Do Not Track

⚠️ Note : Le DNT n'est pas obligatoire pour les sites. TutoFacile respecte vos préférences DNT.
