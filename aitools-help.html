❓ Questions fréquemment posées

Comment installer AITools ? +

AITools s'installe facilement en tant qu'extension Chrome :

  1. Visitez le Chrome Web Store
  2. Recherchez "AITools"
  3. Cliquez sur "Ajouter à Chrome"
  4. Confirmez les permissions
  5. C'est prêt ! Commencez à utiliser
Est-ce que AITools est gratuit ? +

✅ Oui ! AITools est 100% gratuit avec toutes les fonctionnalités disponibles. Aucune limite, aucun coût caché.

Est-ce que mes données sont sécurisées ? +

✅ Oui, absolument. AITools :

  • N'enregistre pas vos données personnelles
  • Utilise le chiffrement pour toutes les communications
  • Respecte votre vie privée en priorité
  • N'affiche pas de publicités intrusives
Comment désactiver un outil sans le désinstaller ? +

Allez dans Paramètres et utilisez les bascules individuels pour activer/désactiver chaque outil selon vos préférences.
