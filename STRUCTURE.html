🎨 Système de Couleurs (CSS Variables)

Variable Couleur Usage Exemple
--primary #0069f6 Bleu vif principal Boutons, headers, accents
--accent #00c4a7 Teal secondaire Accents, hover states
--success #22c55e Vert succès Messages positifs, validations
--warning #f59e0b Orange alerte Avertissements, info
--danger #ef4444 Rouge erreur Erreurs, suppression
--text #1a1a1a Texte sombre Textes, paragraphes
--text-light #666 Texte gris Descriptions, metadata

Dark Mode

Classe: .dark-mode sur html

  • Background: Gradient #0a0a0a → #1a1a1a
  • Texte: #f0f0f0 (blanc)
  • Borders: #333
  • Cards: Gradient #1a1a1a → #252525
