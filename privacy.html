🔒 Politique de Confidentialité

Dernière mise à jour : Janvier 2026

1. Introduction

TutoFacile ("nous", "notre") s'engage à protéger votre vie privée. Cette politique explique comment nous collectons, utilisons et protégez vos données personnelles.

2. Données Collectées

Informations que nous collectons :

  • Adresse IP : Pour améliorer la sécurité et l'analytics
  • Cookies : Pour la préférence de langue (localStorage)
  • Pages visitées : Via Google Analytics pour améliorer le service
  • Données AdSense : Google AdSense collecte des données d'intérêt
  • Contributions : Si vous publiez un tutoriel, nous stockons le contenu

3. Utilisation des Données

Vos données sont utilisées pour :

  • Améliorer l'expérience utilisateur et les performances du site
  • Analyser l'utilisation du site avec Google Analytics
  • Afficher des annonces pertinentes via Google AdSense
  • Prévenir les abus et la fraude
  • Respecter les obligations légales

4. Partage des Données

TutoFacile ne vend jamais vos données personnelles. Cependant, vos données peuvent être partagées avec :

  • Netlify (hébergeur) : Logs de serveur, sécurité
  • Google Analytics : Données d'utilisation anonymisées
  • Google AdSense : Pour servir les annonces

5. Sécurité des Données

Nous utilisons des mesures de sécurité appropriées :

  • HTTPS/SSL pour tous les transferts de données
  • Netlify CDN pour la protection DDoS
  • localStorage pour les données de session (aucun serveur)
  • Aucun stockage de mots de passe en clair

6. Vos Droits RGPD

Conformément au RGPD, vous avez le droit de :

  • Accéder : À vos données personnelles
  • Rectifier : Corriger vos données inexactes
  • Supprimer : Demander l'oubli (droit à l'oubli)
  • Restreindre : Le traitement de vos données
  • Porter : Recevoir une copie de vos données
  • Vous opposer : Au traitement des données

Pour exercer ces droits, contactez-nous à djangogo33.tdac@gmail.com

7. Durée de Conservation

  • localStorage : Jusqu'à suppression manuelle par l'utilisateur
  • Analytics : 26 mois maximum
  • Logs serveur : 90 jours
  • Tutoriels : Indéfiniment sauf suppression demandée

8. Politique de Cookies

Nous utilisons des cookies pour :

  • Stocker votre langue préférée (localStorage)
  • Améliorer l'expérience utilisateur
  • Google AdSense utilise des cookies tiers

Vous pouvez désactiver les cookies dans les paramètres de votre navigateur.

9. Modifications de cette Politique

TutoFacile peut mettre à jour cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.
